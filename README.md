# MindTunes - project

MindTunes is a Java application that acts as a mental health tracker by helping users improve their emotional well-being by providing suggestions based on their current mood.

This system takes input about user's mood and then suggests them music, exercise, diet and ideas to relax themselves

---

## Features

* It gives mood-based suggestion system
* It recommends music according to user's mood
* Gives wellness and relaxation tips
* It is a simple and structured Java implementation
* Has organized modular code (controller, data, service layers)

---

## How does MindTunes Work

1. User inputs their current mood (e.g., sad, happy, calm)
2. The system processes the mood using 'conditional' logic
3. Based on the mood, we give suggestions :

   * Activities (e.g., walk, journaling)
   * Music recommendations
   * Wellness advice
